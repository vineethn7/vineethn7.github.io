The Portal Sleuth Github

The Portal Sleuth is a Django-based Web application that uses RAKE-NLTK to mine user opinions/reviews and evaluate websites. The system takes opinion of various users, based on the opinion; system will decide whether the website is genuine or not. The system uses opinion mining methodology in order to achieve desired functionality.
Languages Used: Python, Django, HTML, CSS

LabourAdda Github

LabourAdda(Android application with Firebase Real time database): I have worked in a 4 member team to develop an android app using Android studio and Firebase realtime database. This app aims in creating a bridge between skilled labourers and contractors so that labourers can find their daily job easily and also the contractors can hire skilled labourers for their work saving time and cost for both the parties. Labourers won't have to wait the whole day at a place searching for work.
Languages Used: Java, AndroidStudio, XML
